# Kanban

Introducing my Kanban app, the perfect solution for managing daily tasks with ease. With the ability to add and categorize to-do lists, this intuitive app allows for quick task prioritization and completion. The Kanban board interface enables you to easily move tasks between different stages of completion, helping you stay organized and on top of your tasks. With this app, you can track progress, visualize workflow, and stay focused on what's important. Give it a try and see how it can help you manage your tasks more efficiently!

## How did I make this app?

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
